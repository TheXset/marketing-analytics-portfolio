# Marketing & Product Analytics Case

## Стек
PostgreSQL · Python (pandas, matplotlib) · Jupyter Notebook

## Структура
- `sql/` — SQL запросы
- `python/` — Jupyter ноутбук
- `ab_test/` — разбор A/B теста
- `data/` — датасеты

## Задачи проекта

**SQL:**
- Рассчитать 7-дневный retention rate когорты пользователей мобильного приложения
- Вычислить медианные месячные траты постоянных клиентов без встроенной функции медианы

**Python:**
- Посчитать уникальных пользователей и сессии по источникам трафика в разрезе платформ (iOS/Android)
- Агрегировать количество пользователей по городам

**A/B тест:**
- Провели A/B-тест email-рассылки: контрольная группа по случайному списку, тестовая по новому алгоритму. Конверсия тестовой чуть выше. Можно ли утверждать что алгоритм стабильно лучше?
