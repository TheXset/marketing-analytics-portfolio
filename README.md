## Задачи проекта

**SQL:**
- Рассчитать 7-дневный retention rate когорты пользователей мобильного приложения
- Вычислить медианные месячные траты постоянных клиентов (без встроенной функции медианы)

**Python:**
- Агрегировать данные о запусках приложения по источникам и платформам (iOS/Android)
- Построить рейтинг городов по количеству уникальных пользователей

**A/B тест:**
- Оценить статистическую значимость результатов email-кампании
- Определить можно ли масштабировать алгоритм подбора аудитории
